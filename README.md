# tri-kanona

A tiny christian booklet for [Три канона: покаянный ко Господу нашему Иисусу Христу, молебный ко Пресвятей Богородице и Ангелу Хранителю](https://azbyka.ru/molitvoslov/tri-kanona-pokajannyj-ko-gospodu-nashemu-iisusu-hristu-molebnyj-ko-presvjatej-bogorodice-i-angelu-hranitelju-cerkovnoslavjanskim-shriftom.html).

Type `make` to build.
